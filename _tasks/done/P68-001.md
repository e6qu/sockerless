## Details

- `PoolConfig`: name, backend_type, max_concurrency, queue_size
- `PoolsConfig`: default_pool + slice of PoolConfig
- `ValidBackendTypes`: 7 entries matching all BackendDescriptor.Driver values
- `ValidatePoolsConfig()`: 8 validation rules
- `LoadPoolsConfig()`: env var → home dir → default fallback
- `DefaultPoolsConfig()`: single "default" pool, memory backend, unlimited
- `GetPool(name)`: find by name, nil if missing
- `PoolNames()`: list all names in order
