# P43-009: Crash Recovery Scanner

**Status:** Done
**Phase:** 43 — Cloud Resource Tracking

## What
Created `CloudScanner` interface and `RecoverOnStartup` function:
- `CloudScanner.ScanOrphanedResources(ctx, instanceID)` — lists tagged resources from cloud
- `CloudScanner.CleanupResource(ctx, entry)` — deletes a cloud resource
- `RecoverOnStartup` — loads registry, scans cloud, registers unknown orphans

Implemented for all 6 backends:
- ECS: ListTasks → DescribeTasks → ListTagsForResource → filter by sockerless-managed
- Lambda: ListFunctions → ListTags → filter
- Cloud Run: ListJobs → filter by labels
- GCF: ListFunctions → filter by labels
- ACA: ListJobs by resource group → filter by tags
- Azure Functions: ListSites by resource group → filter by tags
