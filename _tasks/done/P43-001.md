# P43-001: Shared Tag Builder

**Status:** Done
**Phase:** 43 — Cloud Resource Tracking

## What
Created `backends/core/tags.go` with `TagSet` struct and 3 output formats:
- `AsMap()` — `map[string]string` for AWS/general
- `AsGCPLabels()` — GCP-safe (underscores, truncate 63)
- `AsAzurePtrMap()` — `map[string]*string` for Azure SDK
- `DefaultInstanceID()` — hostname or "unknown"

5 standard tags: `sockerless-managed`, `sockerless-container-id`, `sockerless-backend`, `sockerless-instance`, `sockerless-created-at`.

Added `InstanceID` field to `BackendDescriptor`.

## Tests
6 unit tests in `backends/core/tags_test.go`: AsMap, AsGCPLabels, value truncation, AsAzurePtrMap, DefaultInstanceID, container ID truncation.
