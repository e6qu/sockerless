# P24-004: Add id shell builtin

**Status:** DONE
**Phase:** 24

Added `id` as a shell builtin. Supports `-u` (UID), `-g` (GID), `-un` (username), `-gn` (groupname). Reads from `SOCKERLESS_UID`/`SOCKERLESS_GID` env vars, defaults to 0/0 (root). No flags prints `uid=0(root) gid=0(root)` format.

**Files changed:**
- `sandbox/shell.go` — Added `id` case in builtin switch, added to `builtinCommands` map
- `sandbox/sandbox_test.go` — Added `TestIdBuiltin`
