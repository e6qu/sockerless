# P72-006: GCF Real Execution via Docker API

**Phase**: 72 (Full-Stack E2E Tests)
**Status**: ✅ Complete

## What
Modified GCF backend to invoke the function with the container's command via `X-Sim-Command` header (base64-encoded JSON array). The simulator decodes the header and executes via `sim.StartProcess()`.

Added `invokeWithCommand` helper method. Reads `X-Sim-Exit-Code` response header for exit code.
