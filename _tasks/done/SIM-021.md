# SIM-021: Add CallbackURL config to ECS/CloudRun/ACA backends

**Status:** DONE
**Effort:** S

## What Changed

Added `CallbackURL string` field to ECS, CloudRun, and ACA backend config structs,
loaded from `SOCKERLESS_CALLBACK_URL` env var. When set, backends use reverse agent
mode (agent dials back to backend) instead of forward agent mode (backend dials agent).

## Files Modified

- `backends/ecs/config.go` — Added `CallbackURL` field + env var loading
- `backends/cloudrun/config.go` — Added `CallbackURL` field + env var loading
- `backends/aca/config.go` — Added `CallbackURL` field + env var loading
