# P63-004: Dockerfile VOLUME Directive

**Status:** Done
**Phase:** 63

## Summary
Implemented parsing of VOLUME directive from Dockerfile, supporting both JSON array and space-separated path formats.

## Changes
- `backends/core/build.go`: Added `parseVolumeInstruction` to parse `VOLUME ["/data"]` (JSON) and `VOLUME /data /logs` (space-separated)
- `backends/core/build.go`: Extended `ContainerConfig.Volumes map[string]interface{}` to store volume mounts
- `backends/core/build.go`: Integrated volume directive parsing into build instruction handler
