# P53-004 — Registry Auth Credential Chain

**Status:** Done
**Phase:** 53 — Production Docker API: Code-First

Wired 4-level auth credential chain into image pulls: (1) `X-Registry-Auth` header, (2) `Store.Creds` from `docker login`, (3) `~/.docker/config.json`, (4) anonymous. Added Basic auth parameter to `FetchImageConfig`, `fetchConfigFromRegistry`, and `getRegistryToken`. `decodeRegistryAuth` helper decodes Docker's base64 JSON auth header.
