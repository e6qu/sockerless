Removed VolumeRemove, VolumePrune, ContainerPause, ContainerUnpause overrides from Lambda, GCF, AZF server.go. Deleted corresponding handlers from extended.go files. These were identical to core defaults. FaaS backends now support networks and volumes via core defaults.

## Acceptance Criteria
1. Lambda/GCF/AZF no longer override VolumeRemove, VolumePrune, ContainerPause, ContainerUnpause
2. extended.go files only contain handleContainerPrune (all 3) and handleContainerRestart (GCF/AZF)
