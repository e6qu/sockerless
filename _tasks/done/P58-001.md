# P58-001: Core CI Workflow

Created `.github/workflows/ci.yml` — triggers on push to `main`/`implementation` and PRs to `main`.

Three parallel job groups:
1. **lint** — setup-go 1.24, golangci-lint, `make lint`
2. **unit-tests** — matrix of 5 suites (sandbox, core, bleephub, frontend, integration), all parallel
3. **build-check** — compile 8 binaries to `/dev/null` with `GOWORK=off`

Uses `concurrency` with `cancel-in-progress: true`, matrix `fail-fast: false`.
