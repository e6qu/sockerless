# P43-007: AWS Simulator Tag Support

**Status:** Done
**Phase:** 43 — Cloud Resource Tracking

## What
Added tag storage and retrieval to AWS ECS and Lambda simulators:

**ECS:**
- `ECSTag` struct with Key/Value string fields
- `Tags []ECSTag` on `ECSTaskDefinition` and `ECSTask`
- `handleECSRegisterTaskDefinition`: parse and store tags
- `handleECSRunTask`: parse tags + PropagateTags support (inherits from task definition)
- `handleECSListTagsForResource`: returns actual tags (was returning empty array)

**Lambda:**
- `Tags map[string]string` on `LambdaFunction`
- `handleLambdaCreateFunction`: parse and store tags
- `handleLambdaListTags`: `GET /2017-03-31/tags/{arn}`
- `handleLambdaTagResource`: `POST /2017-03-31/tags/{arn}`
