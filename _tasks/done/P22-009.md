# P22-009: Fetch real image config from Docker Hub registry

**Component:** Backend Core
**Phase:** 22
**Estimated effort:** L
**Status:** DONE

## Description
Created backends/core/registry.go with FetchImageConfig() that fetches real image config from Docker v2 registries (Docker Hub, ghcr.io, gcr.io, etc.). Supports anonymous auth, manifest lists (multi-arch), in-memory caching, and graceful fallback on network errors. Opt-in via SOCKERLESS_FETCH_IMAGE_CONFIG=true.
