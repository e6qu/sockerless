# E2E-007: Makefile targets

**Status:** DONE
**Phase:** 13 — E2E Runner Tests
**Effort:** S

## What Was Built

Added to root Makefile:
- `e2e-github-build` — builds the Docker image for GitHub runner E2E tests
- `e2e-github-%` — runs all workflows against a specific backend (e.g. `make e2e-github-ecs`)
- `e2e-github-all` — runs against all 7 backends
- `e2e-gitlab-%` — runs all pipelines against a specific backend
- `e2e-gitlab-all` — runs against all 7 backends
- `e2e-all` — runs both GitHub and GitLab E2E tests against all backends
