# SIM-022: Add reverse agent to ECS backend

**Status:** DONE
**Effort:** M

## What Changed

Added reverse agent support to the ECS backend. When `CallbackURL` is configured:
- `taskdef.go`: Uses `BuildAgentCallbackEntrypoint` and injects callback env vars
  (`SOCKERLESS_CONTAINER_ID`, `SOCKERLESS_AGENT_TOKEN`, `SOCKERLESS_AGENT_CALLBACK_URL`)
- `containers.go` `handleContainerStart`: IsTailDevNull early return for helper
  containers, Prepare/WaitForAgent/WaitForDisconnect pattern, sets AgentAddress="reverse"
- `containers.go` `handleContainerKill`/`handleContainerRemove`: AgentRegistry.Remove cleanup

## Files Modified

- `backends/ecs/taskdef.go` — Callback entrypoint + env vars, updated function signature
- `backends/ecs/containers.go` — Reverse agent flow in start/kill/remove handlers
